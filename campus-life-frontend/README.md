# 智慧校园生活助手前端

本目录为智慧校园生活助手的前端工程，当前主要完成前端B负责的管理端开发与页面美化工作。

当前版本已根据 `接口文档(V1.3).md` 对管理端接口封装、页面功能和本地假后端进行同步更新。

## 技术栈

- Vue 3
- Vite
- Element Plus
- Vue Router
- Pinia
- Axios
- ECharts

## 启动方式

先启动本地假后端：

```powershell
npm.cmd run mock
```

再打开另一个终端启动前端：

```powershell
npm.cmd run dev
```

访问：

```text
http://localhost:5173
```

管理端登录：

```text
手机号：18800188000
密码：Aa123456
```
